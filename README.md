## Contexto
A partir de una base de datos ya procesada y refinada (aproximadamente el 70% del trabajo de un científico de datos) se deben analizar los escenarios cotidianos de múltiples individuos y como afectan estos a sus emociones. Además de contar con la identificación de cada individuo, sabemos que actividad realizó, el horario de la misma, el tiempo de duración de cada actividad, etc.
A si mismo, poseemos más información ajena al escenario estudiado de cada individuo, ligada a la sociodemografía de cada uno, lo que permitirá cruzar la información y correlacionar los análisis realizados por grupos sociodemográficos.
